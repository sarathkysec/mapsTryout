#  OpenStreetMap Tryout

 It provides a sleek interface for exploring the world with OpenStreetMap. GeoAtlas is a modern, serverless web application built with vanilla JavaScript and Leaflet.js.

## ✨ Features

- **Modern UI**: Built with a glassmorphism design language and premium typography (Outfit).
- **Interactive Map**: Seamless pan and zoom using Leaflet.js and CartoDB Voyager/Dark tiles.
- **Global Search**: Find any location world-wide using the integrated Nominatim API.
- **Dark Mode**: Fully supported with dynamic tile switching for a comfortable night experience.
- **Responsive**: works beautifully on desktops, tablets, and mobile devices.
- **Serverless**: No backend required. Host it anywhere for free.

## 🚀 Getting Started

### Local Development
To run the project locally, you can use any static file server.
```bash
npx serve .
```

### Deployment (Serverless)
Since this project is entirely frontend-based (HTML/CSS/JS), you can deploy it to several serverless platforms for free:

1.  **Vercel / Netlify**: Just connect your repository or drag and drop the folder.
2.  **GitHub Pages**: Push your code to a repository and enable "Pages" in the settings.
3.  **Cloudflare Pages**: High-performance static hosting with an easy CLI.

## 🛠️ Technology Stack

- **Mapping**: [Leaflet.js](https://leafletjs.com/)
- **Geocoding**: [Nominatim API](https://nominatim.org/)
- **Aesthetics**: Vanilla CSS (Custom Properties, Backdrop-filter, Flexbox/Grid)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)
